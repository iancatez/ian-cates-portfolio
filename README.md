# Ian Cates Portfolio

A modern, single-page portfolio website showcasing my work as a Data Engineer.

## Overview

This portfolio features a sleek, dark-themed design with neon green accents and smooth animations. Built as a responsive single-page application, it provides an engaging way to explore my professional background and projects.

## Features

- **Hero Section** - Introduction with animated text and smooth scroll navigation
- **About Section** - Professional background, skills, and expertise
- **Projects Section** - Bento grid layout showcasing featured work with interactive demos
- **Neon Visual Effects** - Flickering neon glow effects on buttons, cards, and UI elements
- **Smooth Animations** - Scroll-triggered animations with reverse effects
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices

## Tech Stack

- Next.js 16 (React)
- TypeScript
- Tailwind CSS
- Framer Motion
- ShadCN UI
